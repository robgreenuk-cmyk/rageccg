## Part 2 — Considered but NOT changed

Items discussed during review that were deliberately left as-is in `rage_cards_v2.json`.


### A. "Junta" vs "Moot" terminology (Run Like Hell)
The original `Text` field used "Junta voting," while the OCR source used "moot voting." The `Errata` field for this card explicitly reads "Replace Moot with Junta." Per 2018 Comprehensive Rules, "Junta" is the umbrella term covering both Moots and Board Meetings. Since Board Meetings are not yet being scripted, and "Moot" is the term needed for backwards-compatible scripting, the decision was made to standardize on "Moot" wherever a choice arises going forward. Applied here; not yet retroactively swept across the rest of the 1,376-card file.

### B. "Silent Strider" singular vs plural (Scouting Mission)
Original `Text` used plural "Silent Striders" in both instances (matching the plural form used elsewhere in the file, e.g. in `Requires` fields on other cards). The OCR source used singular "Silent Strider" in both instances. Decision: use singular in this card's `Text`, since the card requires a specific single-target hook and singular improves scripting readability. This was NOT applied retroactively to other cards' plural usage (e.g. `Requires` fields elsewhere in the file, which still read "Silent Striders").

### C. Image filename mismatches (not corrected)
Three cards were flagged as having `ImageFile` values that do not match the expected naming convention used elsewhere in the file (`rage.image.<category>.<card.name>`). Confirmed real filenames are known but left unchanged in this pass, since this task was scoped to `Text` field truncation only:
- **Silver Claws** — `ImageFile` is `"silverclaws"`; actual file is `rage.image.gift.silver.claws`
- **Spirit of the Fray** — `ImageFile` is `"spiritofthefrey"`; actual file is (assumed) `rage.image.gift.spirit.of.the.fray`, note also the "frey"/"fray" spelling discrepancy
- **Taking the Death Blow** — `ImageFile` is `"takingthedeathblow"`; actual file is `rage.image.combat.event.taking.the.death.blow`

### D. "Gaia Fetish" keyword origin (Sands of Sleep, and other Fetish cards)
The `Keywords` field "Gaia Fetish" appears in several Fetish-type cards (e.g. Sands of Sleep, Bones of Shakir Hind) where the OCR source only ever says "Fetish." Possible early-development naming later refined to just "Fetish," but this is unconfirmed. Left unchanged pending further research.

### E. "Black Fury" singular vs "Black Furies" plural (Body Wrack)
JSON `Requires` field uses singular "Black Fury"; OCR text uses plural "Black Furies." On manual inspection of the file's own conventions, singular "Black Fury" was confirmed to be the correct/consistent form used elsewhere. No change made — this was a false flag, not an actual discrepancy.

### F. Truncated `Errata` fields (left as original — no source available)
A number of cards have `Errata` fields that are themselves truncated mid-sentence, in addition to the `Text` truncation. The OCR document provided only covers card `Text`, not `Errata`/clarification text, so there is currently no source to complete these. Per standing instruction, these are left untouched until a verified source is found — this project deals only in confirmed, sourced text, never inferred completions:
- Distractions
- Rite of Investiture
- Rite of Wisdom
- Lone Wolf Lupo
- Oisin Mac Gaelach

- Tribal War

### G. Stray trailing escaped-quote artifacts in `Errata` fields (left as original)
Several `Errata` fields end with an orphaned `\"` character after the final period, appearing to be a pre-existing artifact in the source file rather than a truncation. Left untouched as out of scope for this pass:

- Jackal's Curse

### H. Repeated phrase in Errata (Tribal War)
The `Errata` field for Tribal War contains a duplicated sentence ("You must name two different tribes." appears twice in sequence). Flagged as a likely pre-existing source typo; not corrected as part of this Text-only pass.
